# Release v0.3.0

Released: June 2026

## Overview

MDK v0.3.0 focuses on extensibility, multi-host deployments, and a richer UI data layer. The headline additions are a formal plugin system for gateway routes, a local discovery mode that bypasses DHT for same-machine setups, an HRPC client transport for cross-host connections, and a graceful shutdown API. On the UI side, a wave of new components covers alerts, inventory, repairs, and operational reporting.

## Highlights

**BE:**
- Plugin system — gateway routes are now delivered by declarative plugins (`mdk-plugin.json` manifest); built-in auth and telemetry plugins ship out of the box, and custom plugins can be loaded at boot via `extraPluginDirs`
- Local discovery mode — workers and Kernel can exchange RPC keys via a shared local directory instead of joining the DHT, enabling single-process and same-machine deployments without network overhead
- HRPC client transport — the client package now supports connecting to a remote gateway over Holepunch RPC in addition to the existing UNIX socket IPC
- Lifecycle API — `onShutdown()` and `shutdown()` provide uniform, graceful teardown across kernel, gateway, and worker handles
- Node.js minimum bumped to `>=24` across all backend packages
- Examples consolidated and expanded under `examples/backend/`, covering multi-process, single-process, Docker, and per-device-family patterns

**UI:**
- Alert components - current and historical alert views powered by new `useCurrentAlertDevices` and `useHistoricalAlerts` hooks
- Inventory management - device inventory table with movement tracking
- Repairs - repair and maintenance log tracking with `RepairLogChanges` page
- Reporting tool - a new revenue report and a consolidated operational dashboard, a site-view hashrate tab, and shared charting improvements (`MinMaxAvg` summaries, configurable chart title slots, and consistent percent formatting), all driven by one shared timeframe model
- New chart primitives - `AverageDowntimeChart`, `ThresholdLineChart`, `OperationsEnergyCostChart`, `MinMaxAvg`

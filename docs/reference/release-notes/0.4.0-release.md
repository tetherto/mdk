# Release v0.4.0

## Overview

v0.4.0 delivers end-to-end write actions, a Pool Manager UI, inventory management, and new chart primitives, alongside a breaking CSS split in the devkit package.

## Breaking changes

- **UI — devkit CSS split**: As part of the performance improvements, `@tetherto/mdk-react-devkit/styles.css` now ships only tokens + core primitives; foundation component styles moved to a new `styles-foundation.css`. Foundation consumers must import both (core first).

## What's new

- **Pool Manager**: Full foundation feature — sites overview, miner explorer, pools list, site detail, actions sidebar, assign-pool modal, 23 adapter hooks, and a CLI scaffold page (`mdk-ui add page PoolManager`). [Blueprint](../../../ui/packages/react-devkit/blueprints/pool-manager.md)
- **Write-action flow**: End-to-end write actions, the ORK gains action management, caller-proxy, and a permissions module; `@tetherto/mdk-react-adapter` ships a matching set of write hooks
- **Inventory & spare-parts management**: `MovementDetailsModal` for historical device movements, plus six spare-parts CRUD modals including CSV bulk upload via `BulkAddSparePartsModal`
- **New chart and input primitives**: added to `@tetherto/mdk-ui-core`
- **Mock control service**: `npm run mock` starts an in-repo mock for Worker development

## Removed

- JetBrains Mono thin weight from `@tetherto/mdk-fonts`
- Legacy `backend/workers/base` and miner scaffolding files

## Fixed

- **Backend** — Kernel (DHT discovery, HRPC transport, IPC listener, telemetry collector, command dispatcher, health monitor); MDK API (`getKernel`, `startWorker`, `startGateway`, `waitForDiscovery`, `startServices`); per-Worker `mdk-contract.json`; gateway MCP endpoint; services module (PM2 / Docker).
- **UI** — package split (`@tetherto/mdk-ui-core`, `@tetherto/mdk-react-adapter`, `@tetherto/mdk-react-devkit`, `@tetherto/mdk-ui-cli`); Zustand stores; React 19; agent-first CLI; catalog app.
- Console errors in the catalog and full-site UI
- Full-site miners local-discovery watch and example setup
